# Re-Mmogo Frontend

Simple React frontend scaffold for the INFS 202 full-stack group project.

## What is included

- React with Vite
- React Router navigation
- Class-based pages and shared components
- Auth context with token storage
- API service layer prepared for an Express backend
- Forms with client-side validation

## Run locally

```bash
npm install
npm run dev
```

## Backend expectation

Set `VITE_API_BASE_URL` to your backend API root, for example:

```bash
VITE_API_BASE_URL=https://we-are-together-backend.onrender.com
```
